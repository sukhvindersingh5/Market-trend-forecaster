### Objective

Build a multi-source data ingestion pipeline to collect consumer sentiment data for smart speaker products:
Amazon Alexa
Google Nest Mini
Apple HomePod Mini

### Data Sources

The dataset combines consumer opinions from multiple platforms:
Amazon product reviews (Kaggle dataset)
YouTube comments (YouTube Data API)
News articles (GNews API)
Web-scraped product reviews (technology review websites)

### Data Processing

Steps performed:
Text cleaning (lowercase, remove noise)
Standardized schema across sources
Duplicate removal
Multi-source merging

### Final Dataset
2,679 reviews/comments
13 news articles
3 products covered
